## Quick Reference

### Level 1: Context Reduction
```bash
/prime                  # Prime with focused context
/mcp-check             # Verify MCP configuration
```

### Level 2: Sub-Agent Delegation
```bash
/delegate [agent] [task]    # Delegate to sub-agent
/review-code [files]        # Quick code review
```

### Level 3: Context Bundles
```bash
/bundle-save [name]         # Save context
/bundle-load [name]         # Load context
```

### Level 4: Multi-Agent Orchestration
```bash
/expert [agent] [task]      # Deep specialist
/parallel [agents] [task]   # Multiple agents
/orchestrate [workflow]     # Full workflow
```
