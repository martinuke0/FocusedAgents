# Expert Agents

Deep domain specialists for Level 4 comprehensive analysis.

## Available Experts
- `security-auditor` - Security vulnerability analysis
- `performance-optimizer` - Performance optimization
- `architecture-reviewer` - System design review
- `api-designer` - REST API design

## Usage
```bash
/expert [expert-name] "[detailed task]"
```

## Creating Custom Experts
```bash
cp -r _TEMPLATE my-expert
# Add comprehensive supporting files
```

## Template
Use `_TEMPLATE/` as foundation for domain-specific experts.
